本地优先的 Markdown 阅读与编辑器。打开文件夹，扫描文档，保存图片资产、追踪链接关系、检查项目健康、一键修复图片路径、搜索和导出都在你的电脑上完成。

Workflow

为本地文档工作流做减法

它不是云笔记，也不是复杂知识库。它专注于把本机 Markdown 文件打开、组织、保存、阅读和导出。

可靠保存

支持保存与另存为、未保存变更提醒，以及 Cmd/Ctrl + S 快捷键，降低桌面写作时误丢内容的风险。

图片资产管理

拖入图片时会按当前文档位置插入真实相对路径，资产面板会展示每张图片的引用关系，未引用图片也能被快速识别。

项目健康检查

打开项目后集中检查缺失图片、未引用图片资产、绝对路径图片、重复标题和失效文档链接，支持定位问题并一键修复绝对路径图片。

新建文档和文件夹

在项目面板直接新建文档或新建文件夹，自动刷新项目树并打开新文档，适合持续维护产品说明和知识库目录。

快速打开

项目中可以按文件名或路径快速筛选并打开文档，频繁切换项目说明、会议记录和章节资料时更直接。

反向链接和失效链接

当前文档会展示哪些项目文档链接到它，也会提示当前文档中无法解析的本地 Markdown 链接；预览区支持本地链接跳转。

写作工具栏

常用标题、加粗、链接、图片、代码、表格和任务项可以快速插入，减少重复输入 Markdown 片段。

专注与导出

专注模式隐藏旁侧面板，保留纯写作空间；完成后可把当前 Markdown 导出为可分享的 HTML 文件。

项目级搜索

打开文件夹后可以跨 Markdown 文档搜索内容，直接跳转到命中文档，适合本地资料库和项目说明。

项目级扫描

选择一个文件夹后自动整理 Markdown 文档，并可调整扫描扩展名、忽略目录和通用过滤规则。

Project Repair

扫描 → 定位 → 修复 → 导出

01

扫描项目

读取本地 Markdown 和图片资产，建立文档、图片和本地链接之间的关系。

02

定位问题

健康检查项可以直接打开对应文档，图片资产面板会列出引用关系和未引用状态。

03

修复路径

绝对路径图片可以一键修复为相对路径，减少把本机私有路径写进交付文档的风险。

04

导出交付

修复后继续阅读、搜索、保存和导出 HTML，保持本地文档工作流闭环。

Local First

文档留在本机

Local Markdown Studio 不要求登录账号，不把文档上传到远端服务。你打开的 `.md` 文件和项目目录都在本机读取，适合处理私有文档、内部说明、学习笔记和离线资料。

它保留桌面应用应有的直接感：双击文件、打开项目、打开最近文件、保存修改、管理图片、追踪链接、搜索内容、跳转文档、导出 HTML。功能边界清晰，启动后就是工作区。

Install

macOS 安装说明

当前分发包未做 Apple Developer 签名和 notarize 公证，macOS 可能把通过飞书或浏览器下载的 App 标记为隔离。

01

打开 DMG

双击 `Local Markdown Studio-macOS.dmg`，把 `Local Markdown Studio.app` 拖入 `Applications`。

02

解除下载隔离

如果提示“已损坏，无法打开”，在终端执行下面的命令。

xattr -dr com.apple.quarantine "/Applications/Local Markdown Studio.app"
03

启动应用

从“应用程序”打开，或继续在终端执行启动命令。

open "/Applications/Local Markdown Studio.app"
