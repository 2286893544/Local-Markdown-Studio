Workflow

为本地文档工作流做减法

它不是云笔记，也不是复杂知识库。它专注于把本机 Markdown 文件打开、组织、阅读和导出。

编辑与预览同步

分屏、编辑、预览三种模式快速切换，写作时保持 Markdown 源码和渲染结果同步。

项目级扫描

选择一个文件夹后自动整理 Markdown 文档，适合项目说明、技术笔记和本地资料库。

目录和搜索

自动提取标题目录，支持层级折叠、平滑跳转和搜索高亮，长文档也能快速定位。

导出 HTML

把当前 Markdown 渲染为可分享的 HTML 文件，适合交付说明、评审材料和归档。

Local First

文档留在本机

Local Markdown Studio 不要求登录账号，不把文档上传到远端服务。你打开的 `.md` 文件和项目目录都在本机读取，适合处理私有文档、内部说明、学习笔记和离线资料。

它保留桌面应用应有的直接感：双击文件、打开项目、搜索内容、导出 HTML。功能边界清晰，启动后就是工作区。

Install

macOS 安装说明

当前分发包未做 Apple Developer 签名和 notarize 公证，macOS 可能把通过飞书或浏览器下载的 App 标记为隔离。

01

打开 DMG

双击 `Local Markdown Studio-macOS.dmg`，把 `Local Markdown Studio.app` 拖入 `Applications`。

02

解除下载隔离

如果提示“已损坏，无法打开”，在终端执行下面的命令。

xattr -dr com.apple.quarantine "/Applications/Local Markdown Studio.app"
03

启动应用

从“应用程序”打开，或继续在终端执行启动命令。

open "/Applications/Local Markdown Studio.app"
