## 使用方式

1. 打开 `产品经理AI助手提示词.md`。
2. 复制“系统角色提示词”到 AI 工具的 system / instructions / 角色设定中。
3. 按具体场景复制对应任务模板，例如行业洞察、方法论匹配、头脑风暴、PRD、竞品分析、周报生成。
4. 如果需要更贴合团队工作流，可以在每个 Prompt 下补充固定业务背景、字段口径、输出表格格式和验收标准。
